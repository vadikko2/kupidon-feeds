# kupidon-feeds

Kupidon feeds management service

## Installation

```bash
python3.12 -m venv venv
make install
```

## Running

### Local run

```bash
make run
```

### Run with docker-compose

```bash
make docker-up
make docker-down
```

## Development environment setup

### Install dependencies

```bash
make dev
```

### Setup `pre-commit`

```bash
make pre-commit
```
